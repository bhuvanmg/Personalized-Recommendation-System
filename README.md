    SETUP INSTRUCTIONS: 

1. INSTALL NECESSARY PYTHON REQUIREMENTS USING PIP 

    -  pip install Flask pymongo langchain langchain-community huggingface-hub requests

2. DOWNLOAD OPEN SOURCE LLM MODEL - "llama-2-7b-chat.Q4_0.gguf" INTO THE MODELS FOLDER.

    DOWNLOAD URL :-  https://huggingface.co/TheBloke/Llama-2-7B-Chat-GGUF/tree/main

3. CLONE THE REPOSITORY

4. OPEN THE TERMINAL AND RUN "python recsys.py"

5. OPEN A BROWSER AND GO TO THE URL SPECIFIED IN THE TERMINAL. 

    - http://127.0.0.1:5000


    INSTRUCTIONS TO INTERACT WITH THE CHATBOT : 

1. REGISTER/LOGIN USERS WITH USER_ID AND PASSWORD.

2. IN THE CHAT INTERFACE TYPE "preferences" TO ADD LIKES/DISLIKES IN MOVIES,SONGS,BOOKS.

3. FOR RECOMMENDATIONS, TYPE APPROPRIATE MESSAGES. EXAMPLE : "recommend some movies/books/songs"

4. FOR FINDING SIMILAR USERS, TYPE "match".

5. GO TO PROFILE TO VIEW USER'S PREFERENCES.
